### Simple JSX

React Unforget can cache the JSX elements and reuse them in the next render. This is useful when you have a large JSX tree and you want to avoid the overhead of creating the same tree again and again. **This eliminates the need to use `React.memo` or `React.PureComponent` to optimize the rendering of the component.**

{`export default function SimpleJSX() {
  return <div>Hello world!</div>;
}

`}
